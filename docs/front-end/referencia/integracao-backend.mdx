# Integração com Backend

## URL base por ambiente

| Ambiente | URL |
|---|---|
| Desenvolvimento | `http://localhost:8080` |
| Produção | `https://extensao-unifil-production-82a8.up.railway.app` |

## Mapa completo de endpoints

### Usuários

| Método | Endpoint | Body | Resposta | Front-end |
|---|---|---|---|---|
| POST | `/usuarios` | `{ nome, email, senha }` | Usuário criado | `TelaCadastroComponent` |
| POST | `/usuarios/login` | `{ email, senha }` | `{ token }` | `TelaLoginComponent` |
| GET | `/usuarios/me` | — | Dados do usuário | `TopbarComponent` |

### Planos de aula

| Método | Endpoint | Body / Params | Front-end |
|---|---|---|---|
| POST | `/planos-aula` | DTO do plano | `PlanoAulaComponent` (criar) |
| PUT | `/planos-aula/{id}` | DTO do plano | `PlanoAulaComponent` (editar) |
| GET | `/planos-aula/{id}` | — | Edição, PDF, Excel, duplicação |
| GET | `/planos-aula/meus-planos` | — | `HomeComponent`, `ListaPlanosComponent` |
| GET | `/planos-aula/disciplina/{disciplina}` | — | `PlanoService` (disponível, não usado na UI atual) |
| DELETE | `/planos-aula/{id}` | — | `ListaPlanosComponent` |

### BNCC

| Método | Endpoint | Params | Front-end |
|---|---|---|---|
| GET | `/habilidades-bncc/buscar` | `?termo=` | `PlanoAulaComponent` |
| GET | `/plano-habilidade-bncc` | — | `PlanoService.listarHabilidades()` |

## Autenticação nas requisições

Endpoints protegidos exigem:

```
Authorization: Bearer <jwt_token>
```

O token é obtido no login e armazenado em `localStorage` com a chave `token`.

## Tratamento de erros

| Código HTTP | Contexto | Comportamento no front |
|---|---|---|
| 401 | Token inválido/ausente | Erro no console; sem redirect automático |
| 409 | Email duplicado no cadastro | Mensagem amigável ao usuário |
| Outros | Falhas genéricas | Mensagem genérica ou `console.error` |

## Exportação (processamento local)

PDF e Excel são gerados **no navegador** após buscar o plano completo via `GET /planos-aula/{id}`:

- **PDF** — `jsPDF` com layout customizado (seções, cabeçalho, quebra de página)
- **Excel** — `xlsx` (SheetJS) com planilha estruturada

Nenhum endpoint de exportação é chamado no backend.
